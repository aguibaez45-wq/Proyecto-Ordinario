# Calculadora Universal en Python

Este proyecto permite resolver y graficar funciones matemáticas utilizando Python, Flask, NumPy y Matplotlib.

## Funcionalidades

* Evaluación de funciones matemáticas
* Generación de gráficas
* Tabla de valores
* Interfaz web

## Cómo ejecutar

1. Instalar dependencias:
   pip install flask numpy matplotlib pandas

2. Ejecutar el programa:
   python app_2.py

3. Abrir en el navegador:
   http://127.0.0.1:5000/
